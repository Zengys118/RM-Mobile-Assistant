# RM Mobile Assistant

一款为 RoboMaster 比赛设计的 Android 辅助工具，用于赛前测试与显示辅助。

---

## ✨ 功能简介

- **装甲板模拟**
    - 全屏显示灯条与数字
    - 点击屏幕切换颜色

- **赛场测试（准备模式）**
    - 文字提示轮播
    - 左 / 右区域点击切换步骤

- **测试步骤编辑**
    - 新增 / 编辑 / 删除步骤
    - 调整顺序并保存
